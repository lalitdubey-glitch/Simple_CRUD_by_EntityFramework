# 🗃️ CRUD with Entity Framework — ASP.NET MVC

A simple CRUD application built with ASP.NET MVC and Entity Framework.

## 🚀 Features
- Full **CRUD** operations (Create, Read, Update, Delete)
- **Entity Framework 6** — Code First / Database First approach
- **MSSQL** as database
- **ASP.NET MVC** (Classic Framework, not Core)

## 🏗️ Tech Stack
| Layer | Technology |
|---|---|
| Framework | ASP.NET MVC (.NET Framework) |
| Language | C# |
| ORM | Entity Framework 6 |
| Database | Microsoft SQL Server |
